An end-to-end Bangla text analytics pipeline for near-duplicate detection and clustering using Approximate Nearest Neighbor search and Locality Sensitive Hashing (LSH) in PySpark.

<!--more-->

Key Features:
- Locality Sensitive Hashing (MinHash) for efficient similarity search
- TF-IDF vectorization for text processing
- Clustering algorithms for text grouping
- Streamlit web application for visualization
- Hadoop integration for big data processing

Tools: PySpark, Python, LSH (MinHash), TF-IDF, Streamlit, Pandas, Numpy, Clustering, Hadoop
