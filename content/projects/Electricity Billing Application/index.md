Built a comprehensive database application for Electricity Billing System using SQL and Oracle APEX that dynamically prepares monthly bills and transactions while maintaining relational database with authorization.

<!--more-->

Key Features:
- Dynamic bill generation and transaction processing
- User authorization and access control
- Relational database design and management
- Oracle APEX web interface
- Database triggers for automated operations

Tools: SQL, Oracle APEX, Database Triggers, PL/SQL
