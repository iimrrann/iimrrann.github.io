<!--more-->

Key Features:
- Multi-threaded architecture for concurrent user access
- Synchronization using mutex locks and semaphores
- Thread-safe ticket booking and seat allocation
- Efficient resource management and scheduling

Tools: Linux Operating System, C Multi-threading, POSIX threads, Mutex, Semaphores
